# Fire Beats Review Checklist

Before merge:

- [ ] Run `npm run check`
- [ ] Run `npm run test:unit`
- [ ] Confirm `ArrangementMoments.test.ts` passes
- [ ] Wire planner into `GeneratorOrchestrator.ts`
- [ ] Verify arrangement OFF has no extra moments
- [ ] Verify melody-only has no extra moments
- [ ] Verify disabled drums stay disabled
- [ ] Verify trap/drop has a stronger entry
- [ ] Verify boom-bap does not get cheesy risers or random silence
- [ ] Verify lo-fi remains subtle

By-ear notes to capture:

- Does the drop feel earned?
- Does the beat ever sound broken or like playback failed?
- Does the pre-drop fill make space or clutter?
- Does the first drop bar hit harder without sounding overcompressed?
